# Zepto-Product-Analysis-using-SQL
This project focuses on analyzing Zepto product data using SQL to generate actionable business insights. The analysis includes data exploration, cleaning, stock analysis, discount analysis, revenue estimation, inventory evaluation, and product categorization. 

Key Insights:
Top discounted products
Out-of-stock premium products
Revenue contribution by category
Best value products based on price per gram
Categories offering the highest discounts
Inventory weight analysis by category

Tools Used:
MySQL
SQL Queries
Data Cleaning Techniques
Aggregation Functions
Conditional Logic (CASE Statements)

Project Objective:
To demonstrate SQL skills through real-world business scenarios involving e-commerce inventory and pricing data.
